# Machine Learning Lab Practicals

This repository contains Machine Learning laboratory experiments implemented as part of the Semester VI curriculum.

## Contents
Each folder corresponds to an individual lab practical:

- Python ecosystem for Machine Learning
- Data preprocessing and statistical analysis
- Principal Component Analysis (PCA)
- Simple Linear Regression
- Decision Tree (ID3 Algorithm)
- Naïve Bayes Classifier
- k-Nearest Neighbour (kNN)
- Support Vector Machine (SVM)
- Cross-validation techniques
- k-Means clustering
- Handwritten digit recognition using ANN
- WEKA toolkit demonstrations

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Note
This repository is intended for academic and learning purposes.
